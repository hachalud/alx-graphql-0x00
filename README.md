# 🧪 Rick and Morty GraphQL API Explorer

A multi-phase learning project focused on mastering **GraphQL**, **React (Next.js)**, and **Apollo Client** using the Rick and Morty GraphQL API.

---

## 🚀 Project Overview

This project is designed as a step-by-step journey from basic GraphQL queries to building a fully functional frontend application using modern web technologies.

It is divided into multiple phases:
- `alx-graphql-0x00` → GraphQL fundamentals
- `alx-graphql-0x01` → Advanced queries & pagination
- `alx-graphql-0x02` → Next.js + Apollo Client integration

---

## 🎯 Learning Objectives

### GraphQL Fundamentals
- Write precise GraphQL queries
- Use arguments (`id`, `page`) to filter data
- Understand pagination and schema types
- Avoid over-fetching by selecting only required fields

### Frontend Integration
- Set up a Next.js + TypeScript project
- Configure Apollo Client with GraphQL API
- Use `useQuery` for data fetching
- Manage state and pagination in React components
- Organize project with clean architecture

---

## 🧠 Key Concepts

- **GraphQL Query Language** – Single endpoint for flexible data fetching
- **Schema & Types** – Defines structure of data (Character, Episode, Info)
- **Arguments** – Used for filtering and pagination
- **Pagination** – Handles large datasets using `info.next` and `info.prev`
- **Apollo Client** – Manages GraphQL data, caching, and state
- **TypeScript** – Adds type safety and better development experience
- **React Integration** – Uses `ApolloProvider` and `useQuery`
